# SQL Assignment – ClassicModels Database

This repository contains my SQL assignment completed using the **ClassicModels** 
sample database. The tasks include:

- Basic SELECT & filtering
- JOINs (INNER, LEFT)
- GROUP BY & Aggregations
- ORDER BY & LIMIT
- CASE statements
- Date functions (DATEDIFF, DATE_ADD, DATE_SUB)
- Window Functions (ROW_NUMBER, LEAD)
- Subqueries & CTEs

## Folder Structure

- `/queries` — All SQL solutions
- `/data` — (Optional) database export or schema diagram

## How to Run

1. Import the ClassicModels schema into MySQL Workbench
2. Run the `.sql` files in the `queries` folder
3. Each file includes comments explaining the solution

## Skills Demonstrated

- Relational database design
- Writing analytical queries
- Practical SQL for business analytics
